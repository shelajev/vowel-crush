# Vowel Crush

A simple web-based game where players crush vowels.

## How to Run

0. Open: [https://shelajev.github.io/vowel-crush/](https://shelajev.github.io/vowel-crush/)

1. Clone the repository: `git clone https://github.com/your-username/vowel-crush.git`
2. Navigate to the project directory: `cd vowel-crush`
3. Open `index.html` in your web browser.
